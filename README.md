**GymRivals**

GymRivals is a fitness app that lets users track different types of workouts and stay motivated through friendly competition. The app supports logging lifting sessions, tracking outdoor runs with GPS, and using ML Kit + the device camera to count push-ups or squats in real time. Users can also view past workouts, earn badges, see progress charts, and compete with rivals.

Core Features
- Lifting session tracking
- Run tracking with live GPS + map
- Push-up and squat rep counting (ML Kit + CameraX)
- Rivals system
- Workout history
- Progress charts
- Badges
- Goals
